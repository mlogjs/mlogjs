## Features

- Optimizing compiler
- Easy to use
- Fully Typed API
- Macros
- All Instructions and Variables
- An [online editor]

## Installation

#### In the terminal

```sh
npm i -g mlogjs
```

## Usage

#### Creating `hello.js`

```js
print("Hello world");
printFlush();
```

#### Compile

```sh
mlogjs hello.js
```
