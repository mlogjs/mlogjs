## Features

- Optimzing compiler
- Easy to use
- Fully Typed API
- Macros
- All Instructions and Variables

## Installation

#### In the terminal

```sh
npm i -g mlogjs
```

## Usage

#### Creating `hello.js`

```js
const message1 = getBuilding("message1")
print("Hello world")
printFlush(message1)
```

#### Compile

```sh
mlogjs hello.js hello.mlog
```
